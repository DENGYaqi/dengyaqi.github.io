## Java容器
Java 容器是 Java 集合框架（Java Collections Framework，简称 JCF）的核心组成部分，用于存储和操作一组数据对象。Java 容器提供了灵活的接口和实现类，支持各种数据结构，如列表、集合、映射、队列等。容器通常分为两类：Collection 容器和 Map 容器。
